# Alternate F1 Dashboard

> ⚠️ **This is a dummy/placeholder version.** The full project is still in development.

This repository contains a lightweight static version of our F1 2026 personal dashboard — deployed as a preview while the real project is being built out.

## 🔗 Live Preview

👉 [https://alternate-f1-dashboard.vercel.app/](https://alternate-f1-dashboard.vercel.app/)

## About

This dummy version includes:
- Live F1 2026 standings (via Ergast/Jolpi API)
- Season calendar with countdown to next race
- Last race podium & paddock intel
- 3D circuit hologram viewer (Three.js)
- Broadcast-style editorial design

The **real project** will feature a full backend, user authentication, personalised stats, and a more complete analytics suite.

*For now, this serves as a proof of concept and visual reference.*
